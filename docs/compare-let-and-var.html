問題：letとvarの重複定義

それぞれの即時関数の中で、同じ名前の変数を、letとvarで2回ずつ定義している。実行結果の空欄を埋めよ。

    (function (){
      let a = 1;
      let a = 2;
      console.log(a);
    }());
    // > [①]

    (function (){
      var b = 1;
      var b = 2;
      console.log(b);
    }());
    // > [②]
  
      ① Uncaught SyntaxError: Identifier 'a' has already been declared
      ② 2
