問題：値渡しと参照渡し

実行結果の空欄を埋めよ。

      "use strict";
      function updateNumber(value){
        value = 2;
      }
      function updateKey1(obj){
        obj.key1 = "new";
      }

      let num = 1;
      updateNumber(num);
      console.log(num); // > [①]
      
      let obj = {"key1": "old"};
      updateKey1(obj);
      console.log(obj); // > [②]
    
        ① 1
        ② {key1: 'new'}

        解説
        use strict;を使うことで、曖昧な書き方を禁止したり、処理が高速化されることがあります。
