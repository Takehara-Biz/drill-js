問題：use strict

次のプログラムの各行でエラーが発生するかしないか、する場合はどのようなエラーが発生するか回答せよ。

      (function () {
        hoge1 = 1; // > [①]
      }());
      (function () {
        let num = 010; // > [②]
      }());
      (function () {
        "use strict"
        hoge2 = 1; // > [③]
      }());
      (function () {
        "use strict"
        let num = 010; // > [④]
      }());
    
        ① エラーは発生しない。
        ② エラーは発生しない。
        ③ Uncaught ReferenceError: hoge2 is not defined
        ④ Uncaught SyntaxError: Octal literals are not allowed in strict mode.

        解説
        use strict;を使うことで、曖昧な書き方を禁止したり、処理が高速化されることがあります。
